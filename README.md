# CS 407 Instructor's Demo

This repository contains demonstration code for the class CS 407 Seminar: Computer Graphics at Western Oregon University for Spring term 2024.

## Topics
- 3D graphics using three.js
- Vector and matrix algebra, transformations
- 3D model construction and representation
- Lighting and shading
- Game physics and animation
- a little bit of Javascript web frameworks, using SvelteKit and TypeScript
